collatex_server¶

This module implements the CollateX API for the Capitularia Application Server.

Endpoints¶

POST /collatex/collate¶

Collate versions of one chapter taken from a set of witnesses.

Example request:

POST /collatex/collate HTTP/1.1
Host: api.capitularia.uni-koeln.de
Content-Type: application/json;charset=utf-8

{
    "corresp": "BK.139_1",
    "witnesses": [
        "berlin-sb-phill-1737",
        "berlin-sb-phill-1737?hands=XYZ"
    ],
    "algorithm": "needleman-wunsch-gotoh",
    "normalizations": [
        ""
    ],
    "segmentation": false,
    "joined": false,
    "transpositions": false,
}

Example response:

HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8

{
  "witnesses":["berlin-sb-phill-1737","berlin-sb-phill-1737?hands=XYZ"],
  "table":[
    [ [ {"t": "A",     "n": "a" } ],     [ {"t": "A",      "n": "a" } ] ],
    [ [ {"t": "black", "n": "black" } ], [ {"t": "white",  "n": "white" } ] ],
    [ [ {"t": "cat",   "n": "cat" } ],   [ {"t": "kitten", "n": "kitten" } ] ]
  ]
}
Response Headers

  • Content-Type – application/json;charset=utf-8

Status Codes

  • 200 OK – No errors.

  • 500 Internal Server Error – CollateX reported errors. A report was logged in the server logfile.

  • 504 Gateway Timeout – The CollateX process took too long. Retry later or with less witnesses.

to_collatex(id_, text, normalizations=None)¶

Build the input to Collate-X

Builds the representation for one witness. Returns an object that must be combined into the witnesses array.

Example of Collate-X input:

{
  "witnesses" : [
    {
      "id" : "A",
      "tokens" : [
          { "t" : "A ",      "n" : "a"     },
          { "t" : "black " , "n" : "black" },
          { "t" : "cat.",    "n" : "cat"   }
      ]
    },
    {
      "id" : "B",
      "tokens" : [
          { "t" : "A ",      "n" : "a"     },
          { "t" : "white " , "n" : "white" },
          { "t" : "kitten.", "n" : "cat"   }
      ]
    }
  ]
}
Parameters

normalizations (string[]) – List of string in the form: oldstring=newstring Normalizations applied to each word.

Return Object

The representation of one witness.

tile_server¶

A tile server for Capitularia.

This is a simple OpenStreetMap-style tile server using mapnik to render the tiles.

The probability of the client requesting adjacent tiles is very high. To speed things up, we ask mapnik to render a bigger “metatile”, which we cut into tiles and then cache the tiles. We also add some padding around the metatile, which helps avoiding ugly label placement when a label is near a metatile border.
